# Day_2

## 동등연산자와 일치연산자
동등연산자(==): 암묵적 타입 변환
    - `[1] == 1`의 경우 원시자료형과 참조자료형이더라도 `True`를 반환하기 때문에 권장되지 않는 것.
일치연산자(===): 타입에 대한 엄격한(암묵적 타입 변환이 발생하지 않음) 연산(권장)

## 삼항연산자
condition ? expression1(True) : expression2(False)

## loop
for ([초기문]; [조건문]; [증감문]) {
   `동작 정의`
}

for in과 for of의 차이
- for in: 객체의 열거 가능한 `속성(property)`에 대해 반복
- for of: `반복 가능한 객체`(배열, 문자열)에 대해 반복
- 차이: in은 순서가 없는 경우에도 동작이 가능. of는 순서가 없는 경우 동작이 불가능(반복이 가능하다는 것은 순서가 존재한다는 의미).
- 정리: in은 객체 전용(인덱스의 순서가 중요한 배열에서는 사용하지 않음). of는 iterable 전용.
