# 브레인스토밍
- 원겸: 배우 얼굴인식 -> 필모그래피 리스트업
- 혜령: 애니 또는 포스터 학습 후 소팅
  - 목소리로 연령대, 톤, 성별 등 인적요소 분석 -> 고객센터 또는 영화 큐레이팅 
- 진성:
  - 데이터를 어떻게 얻을 것인가?

- 목소리 데이터를 학습, 개인화된 서비스를 제공할 수 있는 기반을 마련
  ex) 고객센터, 각종 추천서비스
